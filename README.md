# OCT Astronomy Club Website

A responsive website for the October STEM School Astronomy Club.

## Features

- Responsive design
- Dark/Light mode
- Reusable navbar and footer components
- Animated star background
- Smooth scrolling and interactive UI
- Gallery and club information pages

## Built With

- HTML
- CSS
- JavaScript

## Pages

- Home
- About
- Contact
- Join the Club
- Partners

## Run Locally

Clone the repository and open `index.html` using Live Server or any local web server.

```bash
git clone https://github.com/mahmoudgouda419/STEMAstroClub.git
```
